# dotfiles
netoguimaraes' dotfiles

## deps

- `make`
- `git`

## usage

just run `make <rule>`. The available rules are: 

- `openbox`
- `apps`
- `vim`
- `git`
- `bash`

If you are using a non Arch based distro, you'll need to change the `INSTALL_PROVIDER` in the `Makefile` and it's argument in the `app` rule.
